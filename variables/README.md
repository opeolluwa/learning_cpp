# variables

Working with variables, variables can be declared wison

- assignment
- braced initializer
- or function notation

## Using assignment

```c++
#include <iostream>
int main (){
int variable = 45;
}
```

## Using braced initializer

```c++
#include <iostream>
int main(){
int variable {45};
}
```

## Function notation

```c++
#include <iostream>
int main(){

int variable(45);
}
```
